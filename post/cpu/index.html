Computer - CPU(Central Processing Unit) Architecture of the central processing unit (CPU) Different Classes of CPU Registers

cpu

  1. control unit: 从内存中获取指令(instructions)，通过控制寄存器单元和运算单元完成工作

  2. arithmetic unit: 完成实际的计算

  3. memory unit(register):

    1. PC - program counter - 存储当前指令执行的地址(程序是由一系列指令组成)
    2. ACC - accumulator - 存储临时计算结果
    3. MAR- Memory Address Registers: 当前要读写的内存(RAM)数据的地址，(堆栈的地址)
    4. MDR - memory data register:从内存(RAM)读取或者将要写入内存中的数据
    5. IR- Instruction Register : 当前正在执行的指令

    CIR - current instruction register - stores actual instruction that is being decoded and executed

_main: //为main函数建立一个帧，并将stack pointer 写入 ESP 寄存器器
   push   3 // 1.将3写入栈中(将esp寄存器存的stack pointer值取出，减去4个字节（32）位，,将3写入其中，更 Stack Pointer的值
   push   2 // 
   call   _add_a_and_b  // 3.建立新的帧（没有实际的创建操作）
   add    %esp, 8 // 10 stack pointer + 8位，即回收8个地址
   ret

_add_a_and_b:
   push   %ebx // 4. 将寄存器ebx上的值写入栈中（stack pointer值已经增加12）
   mov    %eax, [%esp+8]  // 5. 将  stack pointer + 8位（2）上的值写入寄存器eax中
   mov    %ebx, [%esp+12] // 6. 将 stack pointer +12位（3）上的值写入寄存器ebx中
   add    %eax, %ebx  // 7.ebx和eax上的值相加
   pop    %ebx   // 8.stack pointer + 4位 ,即回收4个位地址
   ret   // 9. 回收当前帧
